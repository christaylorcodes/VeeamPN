# Get-VeeamPNClient

## SYNOPSIS
Will pull a list of existing clients.

## SYNTAX

```
Get-VeeamPNClient [<CommonParameters>]
```

## DESCRIPTION
Will use the web API to get a list of existing clients.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-VeeamPNClient
```

Will return a list of your current clients.
