# Disconnect-VeeamPN

## SYNOPSIS
Clears connection information from memory.

## SYNTAX

```
Disconnect-VeeamPN [<CommonParameters>]
```

## DESCRIPTION
Clears connection information from memory.

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-VeeamPN
```

Removes cached connection information.
